Screen overlay The Security flaw in Android OS.

All you need to know about

Security Firm checkpoint find out a flaw in android, when they contact google, Google says that we knew about this flaw and we are ahead to fix this in Android "O". The Problem is that when you download any app from the PlayStore, that app might be overlay another app means they might be run over another app. Actually Facebook, Messenger and other legitimate apps already uses this feature for notifications. Actually Screen overlay can steal your credentials. Now back to the point, this screen overlay issue was arrived from Android 6.0 Marshmallow but back then google resolve this issue by turned off the screen overlay feature by default and every time some other app want to run over another app it asked the user to authorize. But giving this facility create problems for other legitimate apps like Facebook Messenger, WhatsApp who uses this screen overlays to alerts users to new messages. So, google pull this user authorization feature in Android Marshmallow 6.0.1 and allowed any app downloaded from PlayStore to overlay other apps. Overlays from apps that come from outside Google Play still require the user's OK. That's a problem, because while the vast majority of Android malware comes from "off-road" app markets, malicious apps do make it into Google Play. And thanks to the screen-overlay function, they can capture your credentials pretty easily. That message you think is coming from the app you're using may in fact be from a rogue app that is going to transmit your stolen password back to the bad guys.

So how you can protect your phone from this malware exploit (i.e the screen overlay issue).

For Now since this issue is also unresolved by the Tech-Titans you could at-least Make sure that "Unknown sources" is unchecked in your device's security settings. That will prevent your device from downloading any software from outside the Google Play store. And least by not the last I can say that for now wait for that security update patch from experts which can resolve this issue.
